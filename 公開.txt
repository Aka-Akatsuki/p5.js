p5.js について
p5.js is a JavaScript library that starts with the original goal of Processing, to make coding accessible for artists,
designers, educatots, and beginners, and reineterprets this for today's web

p5.jsは、コーディングをアーティストが利用しやすいようにすることを目的とした、
Processingの本来の目的から始まるJavaScriptライブラリです。
デザイナー、教育者、そして初心者、そして今日のWebのためにこれを再解釈します
